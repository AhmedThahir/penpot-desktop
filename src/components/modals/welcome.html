Welcome to Penpot Desktop

Hi there stranger!

Welcome to Penpot Desktop, communtiy software for you and for your team to design and prototype with. From designing amazing concepts for apps and websites, to collabrating with others and creating amazing products.

If you're selfhosting Penpot, it's possible for Penpot Desktop to switch to it by changing the Hostname in the app settings, which can be accessed by pressing "Ctrl/Command" + "."

Penpot Desktop is currently in alpha stages, so do expect some issues if any occur. If you do by any chance find an issue, please submit them to Penpot Desktop's GitHub. You can also contribute to Penpot Desktop by improving it's quaily or adding new features for everyone to enjoy, by creating a PR on the GitHub repo.

Is this official software by Penpot?

No, this is considered community software. Although we're looking forwarded to becoming official software if possible.

Can I trust Penpot Desktop since it's not official software?

Sure you can! It's open source and the code is public for everyone to see and to investigate if they're suspicious.

Who created Penpot Desktop?

Penpot Desktop was created by Korbs Studio and with the help of contributors.

What if I'm selfhosting Penpot?
