# Installing Penpot Desktop
## System Requirements
**Software**
OS: 
 - Windows 10/11
 - macOS High Sierra and newer
 - Linux
   - Debian 8 and newer
   - Ubuntu 14.04 and newer
   - Fedora 24 and newer

**Hardware**
Memory: At least 4GB of ram, 8GB recommended
